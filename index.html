Bridgify

HTML CSS JavaScript Flask SQLite

Project Overview

Mission: To bridge the communication gap between generations through shared experiences and meaningful digital interaction.

Core Concept: A reciprocal platform where seniors share traditional skills fading from modern society, while youth provide digital skills and mentorship.

Key Impact: Combats social isolation for seniors and provides youth with access to unique life skills and meaningful mentorship.

The Problem & Solution

Challenge: Seniors often face social isolation, while youth lack deep interaction with older generations and over-rely on digital tutorials rather than personal mentorship.

Solution: A sustainable digital ecosystem and centralized system that fosters long-term engagement and community building.

Key Technical Features

  • Skills Market — Browse and manage skills to teach or learn, from cooking to digital literacy.
  • The Journal — A digital storytelling space where seniors share memories via text, photos, or audio.
  • Event Management — Discover and join intergenerational activities with booking calendars and admin tools.
  • Community Groups — Interest-based discussion groups for casual activities and connection.
  • Profile & Privacy — Personalised accounts to track progress, set preferences, and manage visibility.

Outcome & Value

For Seniors: Empowerment through sharing life experiences and connecting with a wider community.

For Youth: Development of empathy and cross-generational understanding through unique mentorship.

Community Goal: Preserving cultural heritage so that no generation is left behind.

Photography Club

HTML5 CSS3 JavaScript Bootstrap 5

Project Overview

Developed as a centerpiece for a digital portfolio, this project involved designing and coding a comprehensive website for a School Photography Club. The primary goal was to create a digital presence that effectively markets the club to prospective members while demonstrating proficiency in modern front-end development standards.

Key Technical Features

  • Responsive Framework — Implemented Bootstrap 5 to ensure the website is fully responsive and adaptive across various devices.
  • Core Web Technologies — Developed using a clean architecture of HTML5, CSS3, and Vanilla JavaScript, strictly avoiding external frameworks to demonstrate core coding logic.
  • Interactive Elements — Features a custom-built membership interest form with a dedicated response page to display user-entered data.
  • Multimedia Integration — Integrated high-resolution image galleries and video content to showcase the club's activities.
  • Structured Navigation — Designed an intuitive navigation menu to allow seamless access across four distinct web pages.

Design & UX Strategy

The project followed a professional design language, focusing on UI/UX principles to ensure high readability and desirability. Files were organized into dedicated folders for CSS, images, and multimedia, maintaining a high standard of professional file management. The site structure was logically planned to address user needs, from club information tables to interactive contact features.

Professional Development

This project was completed alongside the IBM SkillsBuild Web Development Fundamentals, ensuring that development practices aligned with industry-standard badges in UX and Web Design.
